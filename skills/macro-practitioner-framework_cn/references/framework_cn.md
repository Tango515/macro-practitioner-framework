# 宏观实干派框架参考

本参考文件总结了 `macro_investing_framework.md` 中的核心内容，供 skill 调用时快速读取。

## 一句话概括

宏观投资的核心，是识别世界如何从旧均衡偏离，这个偏离如何通过政策、信用、流动性和行为被强化，再找到赔率合适、风险可控的资产表达方式。

## 六个主驱动器

1. 增长
2. 通胀
3. 信用
4. 流动性
5. 政策约束
6. 资本流动

## 分析清单

每次分析都回答以下问题：

1. 当前处于什么范式？
2. 主导因果链是什么？
3. 哪个反馈回路在发挥作用？
4. 哪个环节不可持续？
5. 当前市场共识是什么？
6. 可能的错误定价是什么？
7. 接下来的关键催化剂是什么？
8. 最干净的资产表达是什么？
9. 什么信号验证判断？
10. 什么信号使判断失效？

## 仓位纪律

- 条件足够一致时再放大仓位。
- 不在纯想象上建大仓。
- 不必等所有滞后数据完全确认。
- 如果市场不按预期反应，先减仓再复盘。
- 真正的大钱来自少数高把握窗口，而不是持续高频出手。
